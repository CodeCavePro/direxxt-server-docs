Техническая документация для использования API службы доставки Direxxt


Данная документация была разработана с целью обеспечения возможности сторонним разработчикам cамостоятельно подготовить инфраструктуру необходимую для работы с API службы доставки DIrexxt игнорируя интерфейс, предоставляемый клиентским приложением direxxt24.web.app. Сервис обеспечивает возможность сторонней интеграции со стороны контрагентов для формирования, редактирования и отслеживания доставки посредством отправки HTTP запросов в API службы доставки Direxxt.
