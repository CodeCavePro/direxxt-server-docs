Адреса и краткое описание методов необходимый для интеграции с сервисом службы доставки Direxxt

(Подставьте вместо “SERVER_URL” адрес необходимой API)
Development environment => direxxt-dev.azurewebsites.net
Production environment => direxxt.azurewebsites.net

Authorization

[HTTP POST]
https://SERVER_URL/api/Auth/firebase/authorization Метод для авторизации и получения JWT-токена

Calculation

[HTTP POST]
https://SERVER_URL/api/Calculation/priceLatLng Метод для получения стоимости для предполагаемой доставки используя координаты (Lat Lng).

[HTTP POST]
https://SERVER_URL/api/Calculation/priceString Метод для получения стоимости для предполагаемой доставки используя адрес как строку.

Delivery

[HTTP GET]
https://SERVER_URL/api/Delivery/{id} Метод для получения данных о доставке с указанным id

[HTTP DELETE]
https://SERVER_URL/api/Delivery/delete/{id} Метод для удаления доставки с указанным id

[HTTP GET]
https://SERVER_URL/api/Delivery Метод для получения данных о всех имеющихся доставках

[HTTP PATCH]
https://SERVER_URL/api/Delivery/patch Метод для изменения данных по доставке

[HTTP POST]
https://SERVER_URL/api/Delivery/post Метод для создания доставки
